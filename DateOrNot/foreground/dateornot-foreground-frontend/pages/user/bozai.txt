
登录按钮点击执行 [向后台api发送 this.account]
注册按钮点击执行 [向后台api发送 this.account]

两个方法均需要向后台发送 this.account， 验证账号密码

登录：
	验证账号密码是否正确。向数据库匹配账号密码相等的记录，没有，向前台返回202，登录失败，账号密码错误；有，向前台返回200，登录成功，1-3秒跳转到首页
	
注册：
	验证数据库记录中是否有账号相同的记录，存在，向前台返回203，账号已被使用；没有，向前台返回200，可以注册。然后把注册的账号密码，保存到数据库中





前台向后台发送的数据格式，json
{
	"username":"123123",
	"password":"123123"
}

后台向前台返回的数据格式，json
{
	"status":"success",
	"code":300/301/302/303


}
